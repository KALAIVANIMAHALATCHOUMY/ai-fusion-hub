A modern and professional portfolio showcasing my work in AI, LLMs,
Automation, Web Development, and IoT. This site is designed to highlight
my skills, projects, achievements, and experience in a way that is

This portfolio presents my technical background, hackathon achievements,
and hands-on projects in Generative AI, LLM Agent development, Computer
